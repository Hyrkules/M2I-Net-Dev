Projet de mise en lumière des entreprises locals dans l'alimentaire => Mais aussi des filières d'études dans le domaine 
Importance sur la facilité d'utilisation !! Besoin d'une utlisation même avec connexion internet limitée

2 niveaux : 
 - Commerçants locaux / entreprise
 - Utilisateurs peut être tagué comme chargé de l'entreprise

Carte intéractive
	- Filtre par intérêts (Recherche alternance, Boulangerie, bons plans, marché etc...)
	- Affichage des entreprises locales à proximité
	- Possibilité de cliquer sur une entreprise pour voir les détails rapide + possibilité de voir les détails complets

Page de connexion (Simplifié)
 - Entreprise : Nom de l'entreprise, adresse mail, mot de passe, description de l'entreprise, horaires d'ouverture, photos
 - Utilisateurs : Nom, prénom, adresse mail, mot de passe, objectif d'utilisation ? 

Page de création de compte 
	- Entreprise : Nom de l'entreprise, adresse mail, mot de passe, légitimé de l'entreprise ?
	- Utilisateurs : Nom, prénom, adresse mail, mot de passe, Tag entreprise avec système de validation pour les entreprises

Fil d'actualité intelligent 
	- Mise en avant des entreprises sponsorisées
	- affichage des publications des commerçants locaux
	- Like pour les utilisateurs

Gestion de message
	- Partage des publications
	- Contacter les commerçants


Détail de l'entreprise
	- horaire
	- possibilité de détails pour voir des publications des commerçants
	- Message du commerçant ou description

Publication (Obligatoirement depuis compte entreprise)
	- Titre
	- Description
	- Photo ?
	- Date de publication
	- Lieu sur la carte ?
